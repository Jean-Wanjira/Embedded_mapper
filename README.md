# SIL Embedded Mapper V3.0

A hardware design contribution to the Spatial Intelligence Lab's 
Embedded Mapper project at the African Robotics Unit, University 
of Cape Town.

## Overview

The Embedded Mapper is a compact sensing system that combines 
cameras, GPS, and onboard processing to create visual replicas 
of environments as it moves through them. The latest iteration 
adds onboard GPU processing for machine learning applications, 
with agricultural mapping as a primary use case.

My contribution during a 3 month internship (May to August 2025) 
was focused primarily on hardware design for the V3.0 prototype, 
including integrating a new GPS module and an NVIDIA Jetson Orin 
NX into the existing system architecture.

## My Contributions

**Hardware Design (Primary)**
- Designed and revised the 3D hardware layout in SolidWorks for 
  Mapper V3.0
- Integrated new GPS sensor and NVIDIA Jetson Orin NX into the 
  hardware architecture
- Revised power budget documentation for the updated prototype
- Produced detailed documentation for project handover

**Field Work**
- Supported field tests for data collection and fidelity testing
- Assisted with SLAM based spatial mapping during field sessions

**Software (Exposure)**
- Worked with ROS Humble for system integration
- Python scripting for supporting tasks
